Zero-Gravity Unified Proxy

O proxy definitivo compatível com os binários do Antigravity CLI (agy) e Claude Code. Roteamento inteligente de predições, mapeamento de ferramentas e suporte completo a qualquer provedor de IA com chaves dinâmicas.

🎛️

Proxy Unificado

O mesmo servidor responde nativamente tanto ao Antigravity CLI (/v1internal) quanto ao Claude Code (/v1/messages) na porta local 8084.

🛰️

Múltiplos Backends

Use Nvidia NIM, OpenRouter, DeepSeek nativo, Groq, Mistral ou Ollama local. Suas chaves são resolvidas dinamicamente.

🔑

Header Dinâmico

Mude o modelo e o provedor ativo a qualquer chamada enviando no token do cabeçalho o padrão CHAVE:PROVEDOR/MODELO.
