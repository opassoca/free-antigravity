# 🌌 Free Antigravity — Унифицированный прокси-сервер ИИ с нулевой гравитацией

🌎 **Выберите язык:**

Высокопроизводительный унифицированный прокси-сервер, предназначенный для подключения **Antigravity CLI (agy)** и **Claude Code** к любому провайдеру ИИ (Nvidia NIM, OpenRouter, DeepSeek, Mistral, Groq, Ollama) с динамическим разрешением ключей и интерактивной панелью мониторинга физики невесомости.

---


## 🧠 Как работает маршрутизация моделей

1. **Динамическое обнаружение моделей**: Прокси-сервер автоматически запрашивает активных провайдеров, настроенных в вашем `.env` (Nvidia NIM, OpenRouter и т. д.), кэширует их каталоги моделей с помощью механизма на основе TTL и объединяет их в меню выбора `agy -model` (динамически предлагая более 140 вариантов).
2. **Разрешение плоских ID (Flat ID)**: Плоские идентификаторы, сгенерированные для меню CLI (например, `nvidia-deepseek-ai-deepseek-v4-pro`), прозрачно преобразуются обратно в исходные идентификаторы провайдера (например, `nvidia/deepseek-ai/deepseek-v4-pro`) при запуске диалога.
3. **Ручное переопределение**: Вы по-прежнему можете динамически маршрутировать запросы на конкретные бэкенды, настроив переменные `MODEL_MAP_*` в файле `.env` или передав ключи в формате `ВАШ_КЛЮЧ:ПРОВАЙДЕР/МОДЕЛЬ`.

## 📊 Использование токенов и отслеживание квот

*   **Реальное отслеживание потребления**: Интерцептор прокси-сервера извлекает официальные метрики токенов (`prompt_tokens` и `completion_tokens`), возвращаемые в фрагментах (чанках) ответа потока.
*   **Сохранение статистики**: Данные об использовании сохраняются локально в файле `data/usage_stats.json`.
*   **Фактическое списание квот**: Эндпоинт `/v1internal:retrieveUserQuotaSummary` динамически считывает ваше накопленное потребление и корректно отображает оставшуюся квоту в заголовке CLI `agy`.
*   **API квот в реальном времени**: Отправьте запрос на эндпоинт `/v1internal/quota`, чтобы получить информацию о текущей доле квоты, оставшихся токенах и сводке потребления по всем активным провайдерам и моделям в реальном времени.

## ⚡ Особенности

*   **Унифицированная архитектура прокси:** обслуживает как запросы Antigravity CLI `/v1internal`, так и запросы Claude Code `/v1/messages` на одном локальном порту (`8084`).
*   **Движок мультипровайдеров:** динамическая интеграция с Nvidia NIM, OpenRouter, нативным DeepSeek, Mistral AI, Groq и локальными инстансами Ollama.
*   **Динамические заголовки запросов:** изменение модели и провайдера на лету путем форматирования ваших API-ключей в виде `ВАШ_КЛЮЧ:ПРОВАЙДЕР/МОДЕЛЬ`.
*   **Интерактивная страница невесомости:** премиум-панель в стиле Google, использующая **2D-физику Matter.js**, где буквы бренда и карточки свободно плавают и реагируют на курсор мыши.
*   **Стрим без сбоев:** фрагменты ответа SSE оборачиваются во внутренний конверт ответа Google (`{"response": {...}}`), который ожидает Go-клиент `agy`, что устраняет сбой с нулевым указателем при получении первого фрагмента потока.

## 🚀 Быстрый старт

1.  **Клонировать репозиторий:**
    ```bash
    git clone https://github.com/opassoca/free-antigravity.git
    cd free-antigravity
    ```
2.  **Настроить переменные окружения:**
    Скопируйте файл `.env.example` (из папки `free-claude-code`) и заполните его ключами ваших провайдеров.
3.  **Запустить сервер:**
    ```bash
    python free-antigravity.py
    ```
4.  **Открыть панель управления:**
    Откройте `http://localhost:8084` в вашем браузере.

## ⌨️ Алиасы автопрокси (Необязательно)

Чтобы запускать оригинальные команды из любого каталога без ручного запуска прокси, добавьте эти функции в ваш `~/.bashrc` или `~/.zshrc`:

```bash
# Автозапуск прокси для Claude Code
claude() {
    if ! curl -s -o /dev/null -w "%{http_code}" "http://127.0.0.1:8084/v1internal/" 2>/dev/null | grep -q "200"; then
        python ~/free-antigravity/free-antigravity.py >/dev/null 2>&1 &
        sleep 1
    fi
    CLAUDE_CODE_URL="http://127.0.0.1:8084" command claude "$@"
}

# Алиас для Gemini CLI с использованием обертки
alias agy="~/bin/free-antigravity"
```

Выполните `source ~/.bashrc` для применения изменений.
