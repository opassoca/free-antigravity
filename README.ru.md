# 🌌 Free Antigravity — Унифицированный прокси-сервер ИИ с нулевой гравитацией

Высокопроизводительный унифицированный прокси-сервер, предназначенный для подключения **Antigravity CLI (agy)** и **Claude Code** к любому провайдеру ИИ (Nvidia NIM, OpenRouter, DeepSeek, Mistral, Groq, Ollama) с динамическим разрешением ключей и интерактивной панелью мониторинга физики невесомости.

---

## ⚡ Особенности

*   **Унифицированная архитектура прокси:** обслуживает как запросы Antigravity CLI `/v1internal`, так и запросы Claude Code `/v1/messages` на одном локальном порту (`8084`).
*   **Движок мультипровайдеров:** динамическая интеграция с Nvidia NIM, OpenRouter, нативным DeepSeek, Mistral AI, Groq и локальными инстансами Ollama.
*   **Динамические заголовки запросов:** изменение модели и провайдера на лету путем форматирования ваших API-ключей в виде `ВАШ_КЛЮЧ:ПРОВАЙДЕР/МОДЕЛЬ`.
*   **Интерактивная страница невесомости:** премиум-панель в стиле Google, использующая **2D-физику Matter.js**, где буквы бренда и карточки свободно плавают и реагируют на курсор мыши.

## 🚀 Быстрый старт

1.  **Клонировать репозиторий:**
    ```bash
    git clone https://github.com/opassoca/free-antigravity.git
    cd free-antigravity
    ```
2.  **Настроить переменные окружения:**
    Скопируйте файл `.env.example` (из папки `free-claude-code`) и заполните его ключами ваших провайдеров.
3.  **Запустить сервер:**
    ```bash
    python free-antigravity.py
    ```
4.  **Открыть панель управления:**
    Откройте `http://localhost:8084` в вашем браузере.

### Динамический формат заголовка API-ключа
Отформатируйте свой токен авторизации для динамического переключения бэкендов:
```text
x-api-key: ВАШ_API_ТОКЕН:nvidia/deepseek-ai/deepseek-r1
x-api-key: ВАШ_API_ТОКЕН:openrouter/google/gemini-2.5-pro
```
